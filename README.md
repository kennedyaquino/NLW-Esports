# Plataforma de Encontro de Jogadores

### Projeto

Este projeto foi desenvolvido em um bootcamp na plataforma Rocktseat. A plataforme oferece suporte aos jogadores de games que procuram parceiros para jogar online.

### Tecnologias

- Typescript
- NodeJs
- Prisma
- SQLite
- React
- Tailwind CSS
- Reac Native
- Expo
